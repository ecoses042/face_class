# USAGE_GUIDE.md 문서 설명

## 목적

`USAGE_GUIDE.md`는 학습된 나이 예측 모델을 본인 얼굴 사진으로 직접 테스트하는 방법을 단계별로 안내하는 사용자 가이드입니다.

## 대상 독자

- 코드를 수정하지 않고 모델 결과만 확인하고 싶은 사용자
- 프로젝트를 처음 접하는 사람

## 주요 내용

| 섹션 | 내용 |
|------|------|
| 사전 조건 | 가상환경·체크포인트 존재 여부 확인 방법 |
| 단계별 실행 | 가상환경 활성화 → 사진 준비 → `predict.py` 실행 |
| 출력 결과 | 두 모델의 출력 형식과 MAE 성능 비교 |
| 내부 동작 | 리사이즈 → 얼굴 검출 → ArcFace 임베딩 → FCN 예측 흐름 |
| 문제 해결 | 체크포인트 없음, 얼굴 검출 실패, 모듈 오류 등 대처법 |

## 관련 파일

- `src/predict.py` — 실제 예측 스크립트
- `results/fcn_regressor/checkpoints/best_model.pt` — FCN 체크포인트
- `docs/pipeline.md` — 전체 파이프라인 설명
